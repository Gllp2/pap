Componentes

Antes de montar um PC precisamos de todas as peças necessárias.

Nesta página, mostrarei quais são esses componentes.

Dica: Arraste o cursor sobre cada imagem para ver mais detalhes.

Motherboard

A motherboard é a placa principal do PC. Aqui instalamos o CPU, memória, os cabos para o hard drive e outros, como placas de vídeo e áudio. Também é aqui que recebem-se portas como USB e HDMI.

A PCB* (placa de circuito impresso) das maiores motherboards podem incluir de seis a 14 camadas de fibra de vidro, traços de conexão de cobre e planos de cobre para alimentação e isolamento de sinal. Outros componentes são adicionados à motherboard por meio de slots de expansão. Isso inclui soquetes de processador; módulos duplos de memória em linha; Interconexão de componentes periféricos (PCI), PCI Express (PCIe) e slots M.2 para unidades de estado sólido; bem como conexões de fonte de alimentação.

*É uma placa isolante que na superfície tem uma fina camada de cobre com trilhos condutores que ligam os diversos componentes.

CPU/Processador

A unidade central de processamento (CPU), também conhecida por processador, é inserida dentro de um socket na motherboard. Podemos chamar o CPU de cérebro do PC, pois processa todos os nossos pedidos. Sempre que pressionas uma tecla, clicas com o rato ou abres uma aplicação, envias instruções para o CPU.

A velocidade do processador é medida em megahertz (MHz) e gigahertz (GHz). Um processador mais rápido pode executar instruções mais rapidamente. Mas, a velocidade real do computador depende da velocidade de muitos componentes diferentes – não apenas do processador.

GPU

Uma GPU é um circuito eletrônico usado para processar as imagens e vídeos que vês. Também é necessária para jogos, especialmente se estiverem com os gráficos no "máximo". Se o teu jogo estiver lento, pode ser que a tua placa gráfica não seja suficientemente forte para carregar os gráficos.

Placa de vídeo

RAM

A RAM é a memória de curto prazo do teu sistema. Sempre que o teu computador calcula alguma coisa, armazena temporariamente os dados na RAM até que precise deles. Esta memória de curto prazo desaparece quando o computador é desligado. Se estiveres a trabalhar num documento ou ficheiro, vais precisar salvá-lo para evitar perdê-lo. Quando salvares um arquivo, os dados são gravados no disco rígido, que funciona como armazenamento a longo prazo.

A RAM é medida em megabytes (MB) ou gigabytes (GB). Quanto mais RAM tiveres, mais coisas o teu computador pode fazer ao mesmo tempo. Se não tiveres RAM suficiente, irás perceber que o teu computador fica lento quando vários programas estão abertos. Se esse for o teu caso, tenta adicionar RAM ao teu PC para melhorar o desempenho.

SSD

As unidades de estado sólido (SSDs) são as unidades de armazenamento mais comuns atualmente. Os SSDs são menores e mais rápidos do que as unidades de disco rígido (HDDs). Os SSDs são silenciosos e permitem que os PCs sejam mais finos e leves. As unidades de disco rígido (HDDs) são mais comuns em dispositivos mais antigos.

Fonte de Alimentação

A unidade de fonte de alimentação de um computador converte a eletricidade para o PC. Envia energia através de cabos para a motherboard e outros componentes.

Extras

Placa de Som (Expansão)

A placa de som – também chamada de placa de áudio – é responsável pelo que ouves nas colunas ou fones. A maioria das motherboards têm som integrado, mas podes atualizar para uma placa de som dedicada para obter um som de maior qualidade.

Expansão de placa de som

Placa de Rede (Expansão)

A placa de rede faz com que o teu computador tenha acesso à internet. Pode ser conectada com um cabo Ethernet ou Wi-Fi. A maior parte das motherboards têm conexões de rede integradas e uma placa de rede também pode ser adicionada a um slot de expansão.

Expansão de placa de rede

Bluetooth

Bluetooth é uma tecnologia para comunicação sem fio em curtas distâncias. É frequentemente usada em computadores para comunicação com teclados, mouses e impressoras sem fio. Geralmente é integrado à motherboard ou incluído em uma placa de rede sem fio. Para computadores que não possuem Bluetooth, você pode adquirir um adaptador USB, geralmente chamado de dongle.

Adaptador Bluetooth

Pasta térmica

A principal função da pasta térmica é eliminar entreferros ou espaços (que atuam como isolamento térmico) da área de interface, a fim de maximizar a transferência e dissipação de calor. A pasta térmica é um exemplo de material de interface térmica.
